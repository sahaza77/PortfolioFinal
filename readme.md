# PortfolioFinal

## Présentation

PortfolioFinal est mon portfolio personnel réalisé dans le cadre de mon BTS SIO option SLAM.  
Il a pour objectif de présenter mon parcours, mes compétences techniques, mes projets, mes stages ainsi que ma veille technologique.

Ce portfolio me permet de mettre en valeur :
- mon évolution durant ma formation,
- les technologies que j’ai utilisées,
- les projets réalisés en autonomie ou en équipe,
- mes expériences en stage,
- ma capacité à concevoir une interface web claire et structurée.

## Objectifs du projet

Ce portfolio a été conçu pour :
- présenter mon profil de développeur,
- centraliser mes réalisations,
- préparer l’épreuve orale du BTS SIO,
- servir de support de présentation face au jury,
- valoriser mes compétences auprès de recruteurs ou d’entreprises.

## Contenu du portfolio

Le site contient plusieurs sections :

- **Accueil** : présentation générale
- **Parcours** : formations suivies
- **Compétences** : compétences techniques et outils maîtrisés
- **Réalisations** : projets scolaires et stages
- **Veille technologique** : sujet de veille sur le numérique responsable
- **Conclusion** : synthèse de mon parcours
- **Contact** : moyens pour me contacter
